# TinyApp Project

TinyApp is a light-weight, full stack web application built with Node and Express that allows users to shorten long URLs (Ã  la bit.ly). TinyApp uses encrypted cookies and hashed passwords to ensure user security.

TinyApp was built with Night Owls in mind and uses a dark mode color theme to keep yours eyes nice and relaxed while you browse.

## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command.
- visit "http://localhost:8080/" in your browser.
